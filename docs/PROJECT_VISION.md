# DevVerse AI

## Vision

DevVerse AI is an AI-powered developer platform that helps students and software engineers learn, build projects, prepare for interviews, improve resumes, analyze GitHub profiles, and track coding progress—all from one modern application.

---

## Target Users

- College Students
- Job Seekers
- Software Engineers
- Open Source Contributors
- Developers

---

## Goals

- Learn with AI
- Build projects
- Prepare for placements
- Track DSA
- Improve resumes
- Analyze GitHub
- Practice interviews

---

## Version 1 Features

- Landing Page
- Authentication
- Dashboard
- AI Mentor
- Resume Analyzer
- GitHub Analyzer
- DSA Tracker
